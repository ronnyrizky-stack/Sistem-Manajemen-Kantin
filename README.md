# Sistem Manajemen Kantin

## Description

Sistem Manajemen Kantin merupakan proyek Mata Kuliah Pemrograman Berbasis Objek yang dikembangkan menggunakan Java. Aplikasi ini dirancang untuk membantu pengelolaan menu, pelanggan, pesanan, dan transaksi pada lingkungan kantin secara sederhana dan terstruktur.

## Features

* Manajemen data pelanggan
* Manajemen menu makanan dan minuman
* Pengelolaan pesanan pelanggan
* Perhitungan total transaksi
* Penyimpanan dan pengolahan data objek

## Technologies

* Java
* Object-Oriented Programming (OOP)

## OOP Concepts Applied

* Class and Object
* Encapsulation
* Inheritance
* Polymorphism
* Method Implementation

## Project Structure

* Customer.java
* MenuItem.java
* Order.java
* Kantin.java
* Main.java

## Learning Outcomes

Through this project, I learned how to:

* Design applications using Object-Oriented Programming principles
* Implement relationships between classes and objects
* Manage business logic using Java
* Develop simple management systems based on real-world scenarios
